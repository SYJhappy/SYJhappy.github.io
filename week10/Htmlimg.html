Image label <img> and source attribute (Src)


In HTML, the image is defined by the <img> label.

<img> is an empty tag, meaning that it contains attributes only, and has no closing tag.

To display the image on the page, you need to use the source attribute (SRC). SRC refers to "source". The source attribute value is the address of URL image.

The definition of image grammar:



URL refers to the position of the image storage. If the name is "boat.gif" image in www.w3school.com.cn images directory, then the URL for http://www.w3school.com.cn/images/boat.gif.

The browser will display the image image tags appear in the document. If you would be among the image tag placed in the two paragraph, then the browser will first show the first paragraph, and then display pictures, finally display second.



The replacement text attribute (Alt)


The alt attribute is used to image a definition of prepared alternative text. Replace the text attribute value is defined by the user.



Unable to load the image in the browser, the replacement text attribute tells the reader that they lose information. At this time, the browser will display the alternative text instead of images. For all the images on the page with the replacement text attribute is a good habit, it is helpful to display information, and is very useful for those who use the plain text browser.

Notice:In HTML, in order to display the correct webpage, we need to load all the pictures of the webpage, while loading the picture takes time. So my suggestion is: use the picture carefully.
