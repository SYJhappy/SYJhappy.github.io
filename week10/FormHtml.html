The HTML form is used to collect user input.

The HTML form contains <em> the form elements </em> .

Form element refers to input elements and different types of check boxes, radio buttons, and so on the submit button.

The <input> element

<em> The <input> </em>element is the most important <em> form elements</em> .

<input> elements have many forms, depending on the properties <em> of type</em> .

This is the type used in this chapter:


Type Describe
Text The conventional definition of text input.
Radio The definition of the radio button input (select one of the plurality of choice)
Submit The definition of the submit button (submit form)
