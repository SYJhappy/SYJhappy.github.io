What is HTML?   






  • It stands for Hyper Text Mark-up Language.
  • It is not a programming language, but it is a markup language(A set of mark tags).
  • It uses mark tags to describe the website.



    • What's the HTML tag? 






  • HTML tags are keywords surrounded by angle brackets.
  • HTML tags usually appear in pairs.
  • The first tag in the tag pair is the start tag, and the second tag is the end tag.
  • Start and end tags are known as open tags and closed Tags.
